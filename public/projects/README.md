# Project screenshots

Drop your website screenshots here. Use these exact filenames (the cards look for them):

| File                      | Project        |
|---------------------------|----------------|
| `ishita-music.jpg`        | Ishita Music   |
| `life-in-a-lens.jpg`      | Life in a Lens |
| `varun-and-co.jpg`        | Varun & Co.    |

- Recommended size: **800 × 600** (4:3), JPG or PNG.
- Until a file exists, the card shows its colored placeholder block.
- To change a filename, edit the `shot:` value in `src/components/Projects.jsx`.
